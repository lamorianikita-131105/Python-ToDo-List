
# 📝 Python To-Do List CLI App

A simple and beginner-friendly To-Do List application built using Python.  
This is a command-line (CLI) based project that allows users to manage daily tasks easily.

---

## 🚀 Features

- ✅ Add new tasks  
- 📋 View all tasks  
- ❌ Remove tasks  
- 🎨 Colorful and clean CLI interface  
- 🔄 Interactive menu system  

---

## 🛠️ Technologies Used

- Python 🐍  
- Colorama (for colored output)
